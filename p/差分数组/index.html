差分数组

什么是差分数组？ 差分数组是一个和原数组大小的数组，存储的是相邻数组的差值。

具体来说：

  • 给定一个原数组 arr，其长度为 n。
  • 我们可以构建一个差分数组 diff，其长度也为 n。
  • 差分数组的元素定义如下：
    • diff[0] = arr[0] （第一个元素没有前一个元素，所以它就是原数组的第一个值）
    • diff[i] = arr[i] - arr[i-1] （对于 i 从 1 到 n-1）

一个简单的例子

假设我们有原数组： arr = [8, 2, 6, 3, 1]

我们来构建它的差分数组 diff：

  • diff[0] = arr[0] = 8
  • diff[1] = arr[1] - arr[0] = 2 - 8 = -6
  • diff[2] = arr[2] - arr[1] = 6 - 2 = 4
  • diff[3] = arr[3] - arr[2] = 3 - 6 = -3
  • diff[4] = arr[4] - arr[3] = 1 - 3 = -2

所以，差分数组为： diff = [8, -6, 4, -3, -2]

它有什么用？（核心优势）

差分数组最强大的功能是能极其高效地对原数组的某个区间进行批量增减操作。

场景：如果你想给原数组 arr 中从索引 L 到索引 R 的每一个元素都加上一个值 val。

传统做法：你需要一个循环，从 L 遍历到 R，逐个给元素加上 val。如果区间很大，这个操作的时间复杂度是 O(N)。

使用差分数组的做法：你只需要修改差分数组中的两个值！

  1. 给 diff[L] 加上 val。
  2. 给 diff[R+1] 减去 val。（如果 R+1 超出了数组范围，则忽略第二步）

这个操作的时间复杂度是 O(1)，与区间长度无关！

为什么这样可行？（原理）

让我们通过上面的例子来理解。

原数组： arr = [8, 2, 6, 3, 1] 差分数组： diff = [8, -6, 4, -3, -2]

任务：我们想给 arr[1] 到 arr[3]（即 [2, 6, 3]）的每个元素都加上 5。

传统做法： arr 变为 [8, 7, 11, 8, 1]

差分数组做法：

  • L = 1, R = 3, val = 5
  • 第一步： diff[1] += 5 -> diff[1] 从 -6 变为 -1
  • 第二步： diff[R+1] = diff[4] -= 5 -> diff[4] 从 -2 变为 -7
  • 新的差分数组为： diff_new = [8, -1, 4, -3, -7]

现在，我们来验证这个新的差分数组是否对应新的原数组： 通过差分数组还原原数组（这个过程叫做“前缀和”运算）：

  • new_arr[0] = diff_new[0] = 8
  • new_arr[1] = new_arr[0] + diff_new[1] = 8 + (-1) = 7
  • new_arr[2] = new_arr[1] + diff_new[2] = 7 + 4 = 11
  • new_arr[3] = new_arr[2] + diff_new[3] = 11 + (-3) = 8
  • new_arr[4] = new_arr[3] + diff_new[4] = 8 + (-7) = 1

得到的新原数组是： [8, 7, 11, 8, 1]，这与传统方法得到的结果完全一致！

原理分析：

  • diff[L] += val：这个操作意味着从 arr[L] 开始，之后的所有元素都在原始值的基础上增加了 val（因为还原时是累加差分数组）。
  • diff[R+1] -= val：这个操作是为了抵消掉从 arr[R+1] 开始之后所有元素不应该增加的 val，从而将增加操作精确地限制在 [L, R] 区间内。

/**
 * 从原数组构建差分数组
 * @param arr 原数组
 * @returns 差分数组
 */
function buildDiffArray(arr: number[]): number[] {
    const n = arr.length;
    const diff: number[] = new Array(n);
    
    // 第一个元素特殊处理
    diff[0] = arr[0];
    
    // 从第二个元素开始，计算差值
    for (let i = 1; i < n; i++) {
        diff[i] = arr[i] - arr[i - 1];
    }
    
    return diff;
}

// 示例
const originalArray = [1, 3, 2, 5, 8];
const diffArray = buildDiffArray(originalArray);
console.log("原数组:", originalArray); // [1, 3, 2, 5, 8]
console.log("差分数组:", diffArray);    // [1, 2, -1, 3, 3]

/**
 * 对原数组的某个区间进行增减操作（通过差分数组）
 * @param diff 差分数组
 * @param left 区间左边界（包含）
 * @param right 区间右边界（包含）
 * @param value 要增加的值（可为负数）
 */
function rangeUpdate(diff: number[], left: number, right: number, value: number): void {
    const n = diff.length;
    
    // 在左边界加上值
    diff[left] += value;
    
    // 在右边界+1的位置减去值（如果存在）
    if (right + 1 < n) {
        diff[right + 1] -= value;
    }
}

// 使用示例
const arr = [1, 3, 2, 5, 8];
console.log("原始数组:", arr); // [1, 3, 2, 5, 8]

const diff = buildDiffArray(arr);
console.log("差分数组:", diff); // [1, 2, -1, 3, 3]

// 给索引 1 到 3 的元素都加上 4
rangeUpdate(diff, 1, 3, 4);
console.log("更新后的差分数组:", diff); // [1, 6, -1, 3, -1]

const updatedArray = getPrefixSum(diff);
console.log("更新后的数组:", updatedArray); // [1, 7, 6, 9, 8]

/**
 * 从差分数组求前缀和，还原原数组
 * @param diff 差分数组
 * @returns 原数组
 */
function getPrefixSum(diff: number[]): number[] {
    const n = diff.length;
    const result: number[] = new Array(n);
    
    // 第一个元素保持不变
    result[0] = diff[0];
    
    // 累加前缀和
    for (let i = 1; i < n; i++) {
        result[i] = result[i - 1] + diff[i];
    }
    
    return result;
}

function carPooling(trips: number[][], capacity: number): boolean {
    let n:number = 0;
    for (let i = 0; i < trips.length; i++) {
        const [,,to] = trips[i];
        if (to > n) n = to;
    }
    let diff = new Array(n+1).fill(0);
    //差分数组初始全是0，直接更新相当于构建
    for (let i = 0; i < trips.length; i++) {
        const [num,from,to] = trips[i];
        diff[from] += num;
        diff[to] -= num;
    }
    // 前缀和构建原数组 看有没有超容量的站
    let sum = 0;
    for (let i = 0; i <= n; i++) {
        sum += diff[i];
        if (sum > capacity) return false;
    }
    return true;
};

var carPooling = function(trips, capacity) {
    const d = new Map();
    for (const [num, from, to] of trips) {
        d.set(from, (d.get(from) ?? 0) + num);
        d.set(to, (d.get(to) ?? 0) - num);
    }

    let s = 0;
    for (const k of [...d.keys()].sort((a, b) => a - b)) {
        s += d.get(k);
        if (s > capacity) {
            return false;
        }
    }
    return true;
};

var minNumberOperations = function(target) {
    let ans = target[0];
    for (let i = 1; i < target.length; i++) {
        ans += Math.max(target[i] - target[i - 1], 0);
    }
    return ans;
};
